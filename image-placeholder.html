🖼️ Image Setup Instructions for PureFog Website

📋 Quick Setup

To display your machine photos on the website, you need to add 4 image files to the same folder as your HTML file:

Required Image Files:
📁 Your Website Folder/
├── index.html
├── styles.css
├── script.js
├── 1.jpg ← TH-130 machine photo
├── 2.jpg ← SS400F machine photo
├── 3.jpg ← FONY machine photo
└── 4.jpg ← TH-250 machine photo

🔧 Step-by-Step Instructions:

  1. Rename your machine photos:
    • TH-130 machine photo → rename to 1.jpg
    • SS400F machine photo → rename to 2.jpg
    • FONY machine photo → rename to 3.jpg
    • TH-250 machine photo → rename to 4.jpg
  2. Copy the renamed images to the same folder where your index.html file is located
  3. Open index.html in your web browser to see the images displayed

📐 Image Recommendations:

  • Format: JPG, PNG, or WebP
  • Size: Recommended width: 800-1200 pixels
  • Aspect Ratio: 4:3 or 16:9 works best
  • File Size: Keep under 500KB for fast loading

⚠️ Troubleshooting

If images don't show up:

  • Check that image files are named exactly: 1.jpg, 2.jpg, 3.jpg, 4.jpg
  • Ensure images are in the same folder as index.html
  • Try refreshing your browser (Ctrl+F5 or Cmd+Shift+R)
  • Check that image files are not corrupted

🚀 Alternative: Using Different Image Formats

If you prefer different image formats or names, you can edit the index.html file:

Find lines like <img src="1.jpg" alt="TH-130 Fogger" class="product-photo"> and change the src attribute to match your image file names.

Once you've added the images, your PureFog website will be ready to showcase your professional fogging equipment!
